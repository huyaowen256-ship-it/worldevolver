# WorldEvolver · 前端重设计方案

> Owner: AI Agent | Version: 1.0 | Date: 2026-04-16

---

## 一、产品定位重校准

**产品是什么：**
- 天机宗外门生存互动平台
- 读者以"天机宗外门弟子"身份参与世界演化
- 每日提交行动，午夜批处理裁定，结果写入小说章节

**产品不是什么：**
- 不是"游戏后台"
- 不是"任务管理系统"
- 不是"通用社交平台"

## 二、视觉风格定义

```css
:root {

  --gold-primary: #c9a84c;

  --xuanqing: #4a7a6e;

  /* 危险/陨落：朱砂 */

```

## 三、页面结构重定义

### 3.1 首页（/）

**定位：** 天机宗山门广场，展示世界当前状态

**核心要素：**
1. 天机宗山门标题（金色榜书风格）
2. 当前世界日历：第X年·第X月·第X日（外门弟子历）
3. 当前主线进度：第X章·"章节标题"
4. 今日批处理状态：已裁定 / 待裁定
5. 世界公报（宗门公告栏）
6. 导航令牌：
   - 【修炼道场】→ /dashboard
   - 【外门大比】→ /ranking
   - 【弃徒碑】→ /graveyard（改名）
   - 【作者入口】→ /admin

### 3.2 修炼道场（/dashboard）

**定位：** 玩家个人主页，以"道心镜"展示状态

**核心要素：**
1. 道心镜（天机盘）状态卡：
   - 法号（角色名）
   - 境界：蛰渊境后期
   - 灵脉：六脉天赋（已开四条）
   - 因果等级：⚡ X级
   - 大比排名：第X名
   - 所在区域：外门地字区
2. 今日心诀录入（修炼计划提交）
3. 裁决结果展示（昨日裁定）
4. 修炼日志（个人经历）
5. 宗门公告

### 3.3 外门大比（/ranking）

**定位：** 月度大比榜单，烫金榜样式

**核心要素：**
1. 本月榜单（Top 3 特殊金色样式）
2. 淘汰区标记（最后10名红色警示）
3. 个人排名定位

### 3.4 弃徒碑（/graveyard）

**定位：** 陨落弟子纪念碑

### 3.5 作者后台（/admin）

**定位：** 天机阁（作者的"天机阁"）

**核心要素：**
1. 批处理管理
2. 世界规则配置
3. 章节写作工具
4. 角色/世界状态

## 四、信息架构修正

| 原字段 | 修真化字段 | 说明 |
|--------|-----------|------|
| level | 境界 | 蛰渊境/汲泉境/凝丹境 |
| characterName | 法号 | 修炼者名号 |
| location | 所在 | 外门地字区/荒灵原等 |
| status | 道心状态 | 存活/陨落 |
| karmaRank | 因果等级 | 可继承 |
| rebirthCount | 轮回次数 | 已转生次数 |

---

## 五、技术要求

1. 全站统一使用 jade-scroll 组件类
2. 所有页面应用新的 CSS 变量系统
3. 响应式：mobile-first
4. 禁止使用 emoji 装饰（改用 SVG 图标）
5. 字体：Noto Serif SC（正文）+ 无标题装饰
