# 规格质量自检表 (Specification Quality Checklist)

> 此检查单用于在使用 `/analyze` 审查大纲或设计时，验证故事是否违背了原定核心规格。

## 1. 核心定位检验
- [ ] 故事当前的发展是否仍然维持了“残酷生存流”的基调，还是过度向普通玄幻爽文偏移了？
- [ ] 读者的干预是否导致整个世界法则坍塌或主角降智？

## 2. 主角行为准则检验
- [ ] 叶无痕的行动是否出于极度理智与求存？是否掺杂了无由来的同情感或轻率的冒险？
- [ ] 主角的杀伐决断是否拖泥带水？

## 3. 视角绝对限制（第三人称限定）
- [ ] 章节中是否出现了大段“主角不在场”的人物的内心独白或暗中筹谋？
- [ ] 章节中是否提前解密了连主角都无法推演出的惊天大局？（若有，请全部删除或移到番外纪事）

## 4. 特殊外挂限制
- [ ] 魔族意识碎片是否被当成“老奶奶老爷爷”用来直接和主角无阻碍沟通？（违规：前期只应表现为被动压缩灵气，无交流）

## 5. 世界事件呈现
- [ ] 当世界事件（遗迹、异动等）发生时，主角是否恰好通过合理的自然信息（如公告、任务堂、周围人的议论）知晓，而非强行强加在主角脑子里？
