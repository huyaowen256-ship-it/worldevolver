# 情节逻辑自检表 (Plot Logic Checklist)

> 此检查单用于在使用 `/analyze` 验证正文连载情节时使用。

## 1. 因果闭环检验
- [ ] 本章发生的冲突，是否有前几章的伏笔或合理的动机支撑？
- [ ] 被击败的反派/配角，是否产生了相应的社会关系变动或者后续追杀/影响，还是只是变成了单纯的数据？

## 2. 爽感节奏检验 (按 creative-plan.md 执行)
- [ ] 过去 3-5 章中，是否给出了让读者提神的“小爽点”？（包含但不限于：越阶爆发、获取资源、破解阴谋）
- [ ] 爽点之前，是否做足了足够的心理和物理压力（虐）铺垫？

## 3. 金手指的代价检验
- [ ] 叶无痕从封印中获取好处时，有没有付出代价或者面临失控的风险？
- [ ] “越阶战斗”是否由于信息差或底牌质变导致，逻辑闭环是否圆融？

## 4. 读者的影响验证（沙盒耦合度）
- [ ] 当玩家群体的数值或任务达到阈值触发了局部世界事件时，主角参与该事件的方式是否自然？
- [ ] 没有强行给读者代入的角色赋予“系统外”的高级权力或金手指。
